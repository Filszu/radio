## Environment Variables

To run this project, you will need to add the following environment variables to your .env file

`NEXT_PUBLIC_SUPABASE_URL`
`NEXT_PUBLIC_SUPABASE_ANON_KEY`

`SPOTIFY_CLIENT_ID`
`SPOTIFY_CLIENT_SECRET`

`DB_PASS`
`ADMIN_PASS`

`GOOGLE_ANALYTICS_ID`
